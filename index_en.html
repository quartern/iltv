Welcome to IsraTV (private) Roku App support page

Long story short - this application is kinda old and can no longer be included in the Roku app store. If you still want to use it you have to enable developer mode and "side-load" it.

Here's a good tuorial on side-loading. Follow steps One, through Three in that tutorial but using the following ZIP file in step Three.

» Zipfile for sideload. «

If all goes well the app would now upload and launch. It would (always) appear as the last app in the channel list and at least for now will display "IsraTV - Beta"

Oct 2022 Update:

The above would work upto Roku 11.5.0. If your device has been recently pushed an update wno now you are running Roku OS 11.5 or higher your Roku will tell you that the app has been removed (you may also get a blank screen). For this reason I'm now releasing a super incomplete 2nd generation version of the app, and doing so with a reduced number of channels till I can work some of he kinks. Download this other file and check back often to see if a later version is available.
